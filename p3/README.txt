UNZIP
1. unzip file into a directory. the following files should be included:
    makefile
    README.txt
    smallsh.c
2. enter directory containing the listed files


TO COMPILE:
1. enter the following in the command line and hit enter:
    make

TO RUN:
1. enter the following in the command line and hit enter 
    ./smallsh


The supported commands may be entered
: cd
: status
: exit
